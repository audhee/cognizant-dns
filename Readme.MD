
# Week 1 - Cognizant Digital Nurture 4.0

This folder contains the solutions for the Week 1 exercises from the Cognizant Digital Nurture 4.0 Java Full Stack Engineer program.

The focus of this week is to strengthen Java fundamentals by implementing basic programming concepts, PL/SQL programs, JUnit testing, and introductory design patterns.

## Exercises Completed

### JUnit

- Exercise 1 - Setting up JUnit
- Exercise 2 - Creating Test Classes
- Exercise 3 - Assertions in JUnit
- Exercise 4 - Arrange-Act-Assert (AAA) Pattern, Test Fixtures, Setup and Teardown

### PL/SQL

- Control Structures
- Stored Procedures

### Design Patterns

- Singleton Pattern
- Factory Method Pattern

### Case Studies

- Financial Forecasting
- Inventory Management System
- E-Commerce Search System

## Technologies Used

- Java
- Maven
- JUnit 4
- PL/SQL
- IntelliJ IDEA
- VS Code

## Purpose

The objective of these exercises is to gain hands-on experience with Java development, unit testing using JUnit, database programming with PL/SQL, and basic software design patterns as part of the Cognizant Digital Nurture learning program.
